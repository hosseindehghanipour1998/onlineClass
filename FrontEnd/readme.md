# Vue JS

## Installing
- `npm install`
- `npm install -g vue-cli`
- `vue init <temple-name> <project-name>` 
- `npm run dev`

## Installing Router
- `npm install vue-router`

## Dependencies You Should Install

    "vue": "^2.5.2",
